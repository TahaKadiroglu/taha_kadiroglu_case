Selenium Test Automation with Python (POM & Unittest)

This project is a selenium based test automation framework written in Python. It follows the Page Object Model (POM) design pattern python unittest framework for executing test cases.

Python version: Python 3.10.11
Selenium version: Selenium 4.28.1

To run the tests go to ui_automation/tests directory and execute the command:
python -m unittest test_ui_pages.py -v
