API Test Automation with Python (Requests & Unittest)

This project is an API test automation framework written in Python. It uses the Requests library for API testing and unittest framework for executing test cases.

Python version: Python 3.10.11
Requests library version: 2.32.3

Pet endpoints are used in the from https://petstore.swagger.io/

To run the tests go to 'api_automation' directory and execute the command:
python -m unittest pet_store_api_tests.py -v
